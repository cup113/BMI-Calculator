# readme - BMI计算器的使用说明

### 1. 功能

1. 通过身高与体重计算BMI(体重指数)
2. 通过年龄与性别判断BMI情况（偏瘦/正常/超重/肥胖）

### 2. 注意事项

1. 在“体重”栏内请输入公斤(==kg==)而不是斤(500g)。
2. 在“身高”栏内请输入厘米(==cm==)而不是米(m)。
3. 在“年龄”栏输入6或以上的整数，将==周岁四舍五入==，学生按照(年级数 + 6)填写年龄，将19-22岁（大学），23岁及以上（普通成人）各均使用同一数据。
4. BMI仅以其计算简单、收集信息少而流行，也因此降低了判断的准确率。运动员有时也会超标，但他们的身体状况是正常的。如果你想要准确的结果，建议去医院检查体脂率等指标。
5. 在一天内，一个身材标准的男性（比如身高180cm，体重70kg）一天的BMI最多可以浮动==1==个指数，所以不必纠结，专业检查找专业的人即可

### 3. 操作说明

1. 填写自己的个人信息。输入体重(kg)，身高(cm)与年龄(7+)。
2. 点击“计算”按钮
3. 输出结果。
