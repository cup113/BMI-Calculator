readme - BMI计算器的使用说明

1. 功能

  1. 通过身高与体重计算BMI(体重指数)
  2. 通过年龄与性别判断BMI情况（偏瘦/正常/超重/肥胖）
  3. 通过身高、年龄与性别计算出减肥体重的参考

2. 注意事项

  1. 在“体重”栏内请输入公斤(kg)而不是斤(500g)，否则你会对自己的结果大吃一惊。
  2. 在“身高”栏内请输入厘米(cm)而不是米(m)。
  3. 在“年龄”栏输入6或以上的整数，将周岁四舍五入，学生按照(年级数 + 6)填写年龄，将19-22岁（大学），23岁及以上（普通成人）各均使用同一数据。
  4. BMI仅以其计算简单、收集信息少而流行，也因此降低了判断的准确率。如果你想要准确的结果，建议去医院检查体脂率等指标。
  5. 在一天内，一个身材标准的男性（比如身高180cm，体重70kg）一天的BMI最多可以浮动1个指数，所以不必纠结，专业检查找专业的人即可

3. 操作说明

  1. 填写自己的个人信息。输入体重(kg)，身高(cm)与年龄(6+)。
  2. 点击“计算”按钮
  3. 输出结果。在渐变色条下方会有一个纯色填充指针，BMI即为其指向的数字。渐变条下方的数字代表各分界线的BMI数值。
  4. 下方表格。在这里，你可以看到在其他因素不变时，BMI与体重的关系。你可以从这里参考你需要减肥的重量。

4. 拓展用法

使用下方BMI与体重关系的表格（第1个表格），比如如果你超重了，想要减到正常体重，就可以参照“正常”一行的标准定下目标。

5. 依据

  1. 《国家体质健康标准》（2014年修订版）
  2. 体重指数BMI（百度百科）

6. 代码原理

  1. 存储数据。将各个年龄段正常、超重、肥胖等的指标分界值做记录，存储为数组。
  2. 获取数据并计算。此期间若发现数据异常，则发出警报。
  3. 更新HTML。将数据可视化呈现。
